# Practicas Profesionalizantes II - APAPED
Proyecto final de la tecnicatura superior en desarrollo de software.

!!ATENCIÓN!! Para que el software funcione y conecte correctamente con la base de datos siga los siguientes pasos:

1. Descargue la solución como archivo .zip
2. Descomprima la solución dentro de otra carpeta con el mismo nombre del archivo .zip (es decir: click derecho en el archivo zip y extraer en "Practicas-Profesionalizantes-II---APAPED-master"
3. Puede abrir la solución desde visual studio y ejecutarla o puede encontrar y ejecutar el archivo .exe en la siguiente ruta: "C:\Users\Usuario\Desktop\Practicas-Profesionalizantes-II---APAPED-master\Practicas-Profesionalizantes-II---APAPED-master\Practicas Profesionalizantes II - APAPED\bin\Debug".
4. El software guarda automaticamente una copia de seguridad de la base de datos access (.accdb) en la carpeta documentos al cerrarlo desde el menú del mismo.
